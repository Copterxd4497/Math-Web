## Installation Instructions

### For Frontend:
cd client
npm install

### For Backend:
cd server
npm install

## Running the Application

### Development Mode:
# Terminal 1 - Frontend (port 5173)
cd client
npm run dev

# Terminal 2 - Backend (port 5000)
cd server
npm start

### Production Build:
cd client
npm run build
npm run preview
